## 6. Query Examples

### Example 1: Get all Contacts with their Companies

```sql

  c.id AS contact_id,
  c.data->>'fld_full_name' AS contact_name,
  c.data->>'fld_email' AS email,
  comp.data->>'fld_company_name' AS company_name,
  comp.data->>'fld_industry' AS industry

LEFT JOIN records comp ON c.data->>'fld_company' = comp.id::text
WHERE c.object_id = 'obj_contact'
ORDER BY c.data->>'fld_full_name';
```

## 7. Performance Considerations

### Indexes (Critical!)

```sql
-- 1. Primary queries (object_id + JSONB field)
CREATE INDEX idx_records_object_email
  ON records USING GIN ((data->'fld_email'))
  WHERE object_id = 'obj_contact';

CREATE INDEX idx_records_object_company_name
  ON records USING GIN ((data->'fld_company_name'))
  WHERE object_id = 'obj_company';

-- 2. Denormalized primary_value for list views
CREATE INDEX idx_records_object_primary
  ON records(object_id, primary_value);

-- 3. Relationship lookups
CREATE INDEX idx_relationship_records_from_to
  ON relationship_records(from_record_id, to_record_id);

-- 4. Date range queries
CREATE INDEX idx_records_created_at
  ON records(created_at DESC)
  WHERE object_id IN ('obj_contact', 'obj_opportunity', 'obj_task');
```
